# 📱 WebMed — App de Plano de Saúde

O **WebMED** é um aplicativo de plano de saúde desenvolvido para facilitar o acesso dos usuários aos serviços médicos de forma **rápida e digital**.

Por meio do aplicativo, o usuário pode:

- Gerenciar informações do seu plano
- Acessar serviços médicos importantes
- Cuidar da própria saúde de forma prática

Tudo isso **sem precisar entrar em contato diretamente com o atendimento telefônico**, tornando a experiência mais simples, rápida e acessível.
